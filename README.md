# Professional Portfolio - Kanimozhi A.

A modern, clean, and high-performance personal portfolio tailored for **Software Development** and **AI / ML Engineering** roles.

## 🚀 Produc-Driven AI Focus
This portfolio showcases:
- **Decision Intelligence**: Highlighting the Decision Assembly Platform (DAP).
- **Explainable AI**: Narrative-driven project case studies.
- **Full Stack Expertise**: Backend performance and scalable architecture.

## 🛠️ Technology Stack
- **Frontend**: React.js, Vite
- **Styling**: Vanilla CSS (High-Performance Design System)
- **Deployment**: Optimized for GitHub Pages / Vercel

## 📁 Repository Structure
- `/src/components`: UI modules (Hero, About, Projects, etc.)
- `/src/styles`: Design tokens and global CSS
- `/public`: Static assets

## ⚡ Local Development
1. Clone the repository
2. Install dependencies: `npm install`
3. Run development server: `npm run dev`
